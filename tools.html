What tools are available?

Lin Clark, a very active Drupal contributor, is currently preparing a module that enables microdata and RDF to happily co-exist.

The software library and service any23, which also happens to power the data acquisition module of the semantic indexer Sindice will very soon release support for microdata based on the mapping defined, here.

OpenLink's Virtuoso Sponger supports Microdata natively and turns it into Linked Data.

The upcoming release of TopBraid Composer will include comprehensive support for editing and processing Schema.org Microdata.

Ed Summers is developing an rdflib plugin that enables to parse Microdata and process the resulting RDF.

We're currently working on a microdata-to-RDF gateway for the Schema.org terms using the mapping defined here. It will utilise Ed Summers' rdflib plugin. This tool allows to utilise HTML pages marked up in microdata to be used in the RDF tool chain, such as for SPARQL, reasoning, etc.
