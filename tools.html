What tools are available?

We're currently working on a microdata-to-RDF gateway for the Schema.org terms using the mapping defined here. This tool allows to utilise HTML pages marked up in microdata to be used in the RDF tool chain, such as for SPARQL, reasoning, etc.

Lin Clark, a very active Drupal contributor, is currently preparing a module that enables microdata and RDF to happily co-exist.

The software library and service any23, which also happens to power the data acquisition module of the semantic indexer Sindice will very soon release support for microdata based on the mapping defined, here.
